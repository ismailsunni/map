About

Find shortest road routes between landmarks, or solve the Travelling Salesman Problem (TSP) for up to 8 stops.

How it works

Routes follow real roads using pgRouting — shortest paths on an OpenStreetMap road network.

Stack

OSM → osm2pgrouting → PostgreSQL + pgRouting on Supabase · Dijkstra routing via RPC · Held-Karp TSP (client-side) · OpenLayers
